# ACM HDOJ Solutions

本项目包含杭州电子科技大学在线评测系统（HDOJ）的题目解答。

## 项目特点

本项目中的所有题目解答均使用 **Claude Code** 进行编程和自动提交到 GitHub。

## 目录结构

每个题目解答都存储在独立的子目录中，所有 Java 文件的类名均为 `Main`，通过不同的子目录来区分题目：

```
hdu_acm/
├── A/          # A+B 问题
│   └── Main.java
└── Sum/        # 求和问题
    └── Main.java
```

## 开发工具

- **编程助手**: Claude Code
- **语言**: Java
